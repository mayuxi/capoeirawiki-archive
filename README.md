# CapoeiraWiki dataset

CapoeiraWiki is an open collaborative encyclopedia dedicated to capoeira history, culture, music, movements, biographies, organizations, and related topics.

This release contains an archived dump of CapoeiraWiki for preservation, citation, and research use.
